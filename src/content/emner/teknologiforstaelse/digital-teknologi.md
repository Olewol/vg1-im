# Digital teknologi — Hvordan datamaskiner fungerer

## 🎯 Hva skal du lære?

I dette emnet skal du forstå hvordan en datamaskin er bygget opp og hvordan den fungerer. Du lærer om hardware (maskinvare), software (programvare), hvordan data lagres og behandles, og hvilke muligheter teknologien gir oss.

---

## 📘 Fagstoff

### Datamaskinens oppbygning

En datamaskin består av flere fysiske komponenter som jobber sammen:

- **Hovedkort (motherboard):** Kretskortet som binder alle komponentene sammen
- **Prosessor (CPU):** "Hjernen" i datamaskinen — utfører beregninger og instruksjoner
- **Minne (RAM):** Korttidsminne som lagrer data som er i aktiv bruk
- **Lagring (SSD/HDD):** Lagrer data permanent — filer, programmer og operativsystem
- **Strømforsyning (PSU):** Gir strøm til alle komponentene
- **Skjermkort (GPU):** Håndterer grafikk og bildebehandling

### Hvordan data lagres og behandles

Datamaskiner bruker **binærtall** (0 og 1) — alle instruksjoner og data blir omgjort til en serie med 0-er og 1-ere som prosessoren kan forstå.

- **Bit:** Den minste enheten — 0 eller 1
- **Byte:** 8 bits — kan representere ett tegn (f.eks. bokstaven 'A')
- **Filenheter:** Kilobyte (KB), Megabyte (MB), Gigabyte (GB), Terabyte (TB)

### Programvare (software)

- **Operativsystem (OS):** Styrer maskinvaren og lar programmer kjøre (Windows, macOS, Linux)
- **Applikasjoner:** Programmer som utfører spesifikke oppgaver (nettleser, tekstbehandling, bildebehandling)
- **Drivere:** Programvare som lar operativsystemet kommunisere med maskinvare

### Skyen — lagring og tjenester på nettet

Skyen betyr at data lagres på servere på internett i stedet for lokalt på din maskin. Eksempler:
- **Google Drive / OneDrive** — fillagring
- **Google Docs / Office 365** — programvare som tjeneste
- **Spotify / Netflix** — strømmetjenester

---

## 💡 Praktiske eksempler

**Eksempel 1: Oppstart av PC**
Når du trykker på av/på-knappen:
1. Strømforsyningen starter
2. BIOS/UEFI sjekker maskinvaren
3. Operativsystemet lastes fra SSD/HDD til RAM
4. Innloggingsskjermen vises

**Eksempel 2: Åpne en nettside**
1. Nettleseren sender en forespørsel via nettverkskortet
2. Serveren svarer med HTML, CSS og JavaScript
3. Dataene lastes ned til RAM
4. GPU-en tegner nettsiden på skjermen

- **Naturfag:** Elektronikk og fysikk — hvordan strøm og signaler fungerer
